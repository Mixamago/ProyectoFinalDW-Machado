El presente sitio web ha sido diseñado para el curso de Desarrollo Web en Coderhouse, camada 38015, con la profesora Laura Avalle, a modo de proyecto final. Dicho proyecto ha recibido la ayuda y tutoría de Claudia Torres.

El sitio web simula pertenecer a una tienda uruguaya destinada a la venta de termos, mates y bombillas. Está compuesto por siete páginas en total:

	1. Índice.
	2. Catálogo de productos.
	3. Inicio de sesión ("Mi cuenta").
	4. Registro de cuenta (Accessible solo mediante la página de "Mi cuenta").
	5. Preguntas frecuentes.
	6. Información relacionada a los productos ("¿Sabías que...?").
	7. Error (perteneciente al error 404).

El sitio web es responsive y se ha intentado que responda a resoluciones de celular, tableta y computadora de escritorio.

Se puede llegar al error 404 mediante los logos de las redes sociales, los botones de los formularios, o los botones de más información ubicados en "¿Sabías que...?".
